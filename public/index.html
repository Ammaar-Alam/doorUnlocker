About This Project

This is a personal project I worked on over the summer. The switch at the top really does over or close my door hehe; it sends a command to my proxy server which routes that command to the Arduino IoT cloud server, relaying that command to the arduino. That arduino is connected to a motor driver which then spins a DC motor, reeling a fishing line knotted around my door handle, pulling it down.

For now, I don't really plan to add any more automation, this project was mainly inspired by my very real problem of locking myself out somewhat frequently; although it was really fun to learn how to work with arduinos and circuitry,,, so maybe idk

All the parts I used, including those bought and printed, are all open source on the Github Repo, including the code, so feel free to make one yourself :)
